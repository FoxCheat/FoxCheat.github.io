Софт использует Cлеш команды
help — Все команды FoxCheats! +
Использование: /help команда
Примеры: /help, /help aim
info — Подробная информация про FoxCheats! +
Использование: /bot
Примеры: /info
aim — Настройка аимбота +
Использование: /aim скорость
Примеры: /aim 0.5s, /aim 0.5с
wh — Настройка wallhack +
Использование: /wh цвет
Примеры: /wh blue, /wh standart
rcs — настройка recoil control +
Использование: /rks значение X значение Y
Примеры: /rcs 1s 2s
delete-me — Удаляет ваши данные из нашей базы данных +
Использование: /delete-me
Примеры: /delete-me
